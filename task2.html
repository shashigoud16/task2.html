Structure of a URL

URL(Universe Resource Location)
A URL (Uniform Resource Locator) is made up of several parts, including a protocol, domain, and path. Parts of a URL

  • Protocol

    • The type of communication used, such as http or https

  • Domain

    • The name of the website, which can include a subdomain and top-level domain

  • Path

    • The file name or directory of the requested page

  • Subdomain

    • A part of the main domain that can be used to organize content into sections

  • Top-Level Domain

    • The type of website, such as .com, .net, .org, or .edu

  • Query

    • A set of parameters that can be added to a URL

Waterfall Methodology

The Waterfall Model is a classical software development methodology. It was first introduced by Winston W. Royce in 1970. It is a linear and sequential approach to software development that consists of several phases. It must be completed in a specific order. This classical waterfall model is simple and idealistic. It was once very popular.

Agile Methodology

Agile methodology is a project management framework that breaks down projects into phases, called sprints. It's an iterative process that encourages teams to reflect and improve after each sprint.
